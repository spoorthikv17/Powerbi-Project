# Project Description
This project analyzes the customer data of American Bank using Power BI.  
It provides insights on:
- Customer demographics (Gender, Marital Status, Education)  
- Job Classification  
- Customer outcomes of previous campaigns  
- Distribution by state  
- Total balance analysis  

## Features
- Interactive dashboard with slicers for Contact Type and Quarter  
- KPI Cards: Total Customers, Successful Campaigns, Average Balance  
- Charts: Bar charts, Pie/Donut charts, Gauge  
- Insights included for quick decision making
